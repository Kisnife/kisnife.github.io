时间轴
条约、报刊、文件、会议

  • 1840年 第一次鸦片战争

    • 1841.5 三元里抗英斗争

    • 1842.8.29《南京条约》中英

    • 1843.7《五口通商章程》

    • 1843.10《虎门条约》中英

    • 1844.7《望厦条约》中美

    • 1844.10《黄埔条约》中法

  • 1851年 金田起义 建立太平天国

  • 1853年 天平天国定都天京

  • 1856年 第二次鸦片战争

    • 1858.6《天津条约》

    • 1860《北京条约》

  • 1861年 洋务运动兴起

  • 1864年 太平天国运动失败

  • 1894年 甲午中日战争爆发

    • 1895.4《马关条约》中日

  • 1894年 兴中会成立

    • 宗旨：驱逐鞑虏，恢复中国，创立合众政府

  • 1898年 戊戌变法（百日维新）

  • 1899年 （秋）义和团运动

  • 1900年 9月7日义和团运动失败

  • 1900年 5月28日八国联军侵华战争开始

  • 1901年 4月，清政府设立督办政务处，宣布实行“新政”

    • 9月7日八国联军侵华战争结束

    • 1901年9月《辛丑条约》

      • 义和团运动失败，八国联军攻入北京

  • 1905年 （8月）中国同盟会成立

    • 章炳麟《驳康有为革命书》、邹容《革命军》、陈天华《警世钟》《猛回头》

    • 黄兴——华兴会、蔡元培——光复会、科学补习所、岳王会等

    • 同盟会纲领 - 驱除鞑虏，恢复中华，创立民国，平均地权

    • 1905年 11月《民报》为机关报，发刊词：“民族、民权、民主” 三民主义

  • 1906年 正式废除科举考试

  • 1911年（4月）广州起义（黄花岗起义）（起义失败）

  • 1911年

    • 5月，保路运动兴起

  • 1911年

    • 10月10日，武昌起义

  • 1912年

    • 元旦，中华民国南京临时政府的创立，孙中山就任中华民国临时大总统。

    • 2月，清帝溥仪被迫宣布退位

    • 3月，颁布《中华民国临时约法》是中国史上第一部具有资产阶级共和国宪法性质的法典

    • 4月，袁世凯取得政权后，建立了北洋军阀统治。

  • 1913年

    • 3月 宋教仁被杀、4月 “善后大借款” 引起国民对袁世凯的不满

    • 3月 孙中山从日本回到上海，发动武装反袁的“二次革命”

    • 6月 袁世凯全国发布《通令尊崇孔圣文》

    • 7月 李烈钧在江西湖口通电讨袁。不久，黄兴在南京宣布讨袁

  • 1914年

    • 5月，袁世凯炮制了《中华民国约法》，用总统制取代内阁制

  • 1914年

    • 6月28日，第一次世界大战爆发。这是一场帝国主义战争。

  • 1914年

    • 7月，“二次革命”失败，孙中山逃亡日本，成立中华革命党

  • 1915年

    • 5月，袁世凯同意与日本签订“二十一条”

    • 9月 陈独秀在上海创办《青年杂志》，后改名《新青年》

    • 12月25日，护国运动

    • 12月 袁世凯发布接受帝位申令，定1916年为“中华帝国洪宪元年”，元旦举行登基大典

  • 1916年

    • 6月 袁世凯，卒

  • 1917年

    • 6月 张勋率“辫子军”北上，丁巳复辟，又称张勋复辟、溥仪复辟

  • 1917年

    • 护法运动

    • 11月7日，俄国爆发十月革命，建立了人类历史上第一个社会主义国家。

  • 1918年 4月，毛泽东成立新民学会

    • 5月，鲁迅发表《狂人日记》

  • 1918年

    • 11月11日，德国签订停战协议，第一次世界大战结束，以协约国的胜利结束。

  • 1919年

    • 5月，五四运动爆发

    • 6月10日，释放全部被捕学生，并罢免曹汝霖、章宗祥、陆宗舆的职务

    • 9月、11月，李大钊发表《我的马克思主义观》一文

    • 10月，孙中山将中华革命党改组为中国国民党

  • 1920年

    • 3月，李大钊指导，邓中夏、高君宇等在北京大学成立了马克思主义学说研究会。

    • 5月，陈独秀在上海成立了马克思主义研究会。

    • 8月，成立上海共产主义小组，陈独秀任临时书记，成员有李汉俊、李达等

    • 8月，陈望道翻译了首个中文全译本的《共产党宣言》

    • 10月，李大钊、张申府、张国焘发起成立北京共产党早期组织，李大钊为书记。

    • 11月，上海共产党发起制定了《中国共产党宣言》，社会主义青年团在上海成立，成为共产党的后备力量

    • 11月，李中主持成立了共产党早期组织领导的第一个产业工会——上海机器公会，出版《机器工人》

  • 1921年 7月23日，中共一大在上海法组界召开

    • 8月，中国共产党在上海成立中国劳动组合书记部，这是党领导工人运动的专门机关。

    • 9月，沈定一等在浙江杭州萧县衙前村成立了第一个农民协会

  • 1922年 1月，香港海员罢工

    • 7月，中共二大召开

  • 1923年

    • 元旦，孙中山发表《中国国民党宣言》

    • 1月26日，发表《孙文越飞宣言》，正式确立联俄政策

    • 2月，京汉铁路工人罢工

    • 2月7日，“二七惨案”

    • 6月，中共三大召开

  • 1924年

    • 1月，中国国民党第一次全国代表大会在广州召开

    • 5月，成立中国国民党陆军军官学校（黄埔军校），11月，周恩来担任政治部主任

    • 7月，在广州开办农民运动讲习所，培养了一批农民运动骨干力量

    • 7月，广州纱厂工人罢工的胜利成为工人运动从低潮转向高潮的信号

    • 11月，孙中山应冯玉祥邀请北上，并发表《北上宣言》。主张召开国民会议。

  • 1925年

    • 1月，中共四大召开

    • 5月，以五卅(sà)运动为起点，掀起了全国范围的大革命风暴

    • 6月，广州、香港两地工人大罢工。

  • 1926年

    • 7月，以推翻北洋军阀统治为目标的北伐战争开始

  • 1927年

    • 2月，国民政府外交部同英国方面签订协议收回汉口、九江的英租界

    • 4月12日，蒋介石在上海发动反共政变

    • 7月15日，汪精卫发动反革命政变，国共合作全面破裂，大革命失败

    • 8月7日，中共中央在汉口秘密召开紧急会议（八七会议）

    • 南昌起义

    • 9月9日，秋收起义

    • 三湾改编

    • 创建井冈山革命根据地

  • 1928年

    • 10月，国民党中央常务委员会通过《询证纲领》

    • 10月、11月，毛泽东写下《中国的红色政权为什么能够存在？》和《井冈山的斗争》两篇文章

    • 12月29日，张学良发出通告，宣布“遵守三民主义，服从国民政府，改易旗帜”

    • 12月，立法肯定了广大农民获得土地的权利

  • 1930年

    • 1月，毛泽东《星星之火，可以燎原》

    • 4月，发布第二个土地法，更正错误，保护中国农民的利益

    • 5月，毛泽东《反对资本主义》

    • 8月，中国国民党临时行动委员会正式成立

    • 10月-31年7月，红军反“围剿”作战

  • 1931年

    • 9月18日，“九一八”事变，日本开始侵华战争

    • 11月中华苏维埃第一次全国代表大会在江西省瑞金县叶萍村举行

  • 1932年

    • 4月15日，中华苏维埃共和国临时中央政府宣布对日作战

  • 1934年

    • 10月中旬，长征开始

  • 1935年

    • 日本在华北制造一系列事端，向中国政府提出华北政权“特殊化”的要求。

    • 国民政府实行“攘外必先安内”的妥协方针

    • 1月15-17日，遵义会议（转折点）

    • 5月9日 田汉作词，聂耳谱曲《义勇军进行曲》，推动了进步思想的传播

    • 10月16日，中共中央率陕甘支队与红十五军团会合

  • 1936年

    • 5月，中共中央公开把蒋介石作为联合对象

    • 9月，明确党的总方针是“逼蒋抗日”

    • 10月，红二、四方面军与红一方面军会师，三军会面，长征胜利结束

    • 12月，张学良、杨虎城扣留蒋介石，提出改组南京政府、停止一切内战、召开救国会议等八项主张

  • 1937年

    • 7月7日，七七事变，日本发动全面侵华战争

    • 7月8日，中国共产党号召全中国同胞团结起来

    • 8月，国共两党将红军主力改编为国民革命军第八路军

    • 8月22日，政治局扩大会议，通过了《关于目前形势与党的任务决定》，提出了关于抗日的基本主张

    • 《抗日救国十大纲领》体现了中国人民的根本利益和要求，体现了中国共产党的人民战争路线，是彻底的抗日纲领

    • 9月22日，国民党中央通讯社发表《中共中央为公布合作宣言》

    • 23日，蒋介石承认中国共产党的合法地位，由此国共两党第二次合作为基础的抗日民族统一战线正式形成

  • 1938年

    • 5月-6月，毛泽东发布《论持久战》总结抗战10个月以来的经验

    • 9月-11月，中共延安举行扩大的六届六中全会。毛泽东提出了“马克思主义中国化”的命题

    • 周恩来担任政治部副部长

    • 卢沟桥事变至10月广州、武汉失守，中国抗战属于战略防御阶段

    • 台儿庄战役大捷，我国取得胜利的第一次重大战役是平型关大捷

  • 1939年

    • 1月，国民党五届五中全会决定成立“防共委员会”，标志国民党由片面抗战转为消极抗战

    • 10月，宪政座谈会，批评国民党一党专政

  • 1940年

    • 1939年冬至至1940年春，国民党顽固派发动第一次反共高潮

  • 1941年

    • 1月，国民党顽固派发动第二次反共高潮

    • 3月，同盟会成立

    • 9月，创办光明报

    • 12月，国民党激起大后方人民对蒋介石的强烈不满

  • 1943年

    • 春，国民党顽固派发动第三次反共高潮

  • 1944年

    • 敌后根据地开始向日本发起局部进攻

  • 1945年

    • 春，正面战场发动局部反攻，收复了一些日占领的城市

    • 5月，国名党六大制定了抢夺人民抗战胜利果实、准备发动内战的方针

    • 7月26日，中美英三国发表波茨坦公告，敦促日本无条件投降

    • 8月14日，日本接受波茨坦公告，15日，日本投降

    • 8月14日，20日，23日，蒋介石连发三封电报，邀请中共中央毛主席到重庆进行和平谈判

    • 8月28日，毛主席偕周恩来、王若飞飞赴重庆进行谈判

    • 9月19日，中共中央正式确定了“向北发展，向南防御”的战略方针

    • 10月25日，中国政府在台湾举行受降仪式。中国收回台湾及澎湖列岛，成为抗日战争取得完全胜利的重要标志

  • 1946年

    • 1月10日，国共双方下达停战令，同日，协商会议在重庆开幕

    • 2月10日，国民党破坏“庆祝政协成功大会”制造校场口惨案

    • 3月，国民党六届二中全会上，蒋介石扩大内战，使得中国共产党的工作重点由争取和平民主逐步转向准备自卫战争

    • 6月23日，上海人民团体联合会派出请愿团，赴南京国民党当局呼吁和平。发生了下关惨案

    • 6月26日，国民党军围攻中原解放区，挑起全国性内战

  • 1947年

    • 2月，粉碎了国民党军队的全民进攻

    • 6月，粉碎了国民党对陕甘宁边区和山东解放区的重点进攻

    • 土地改革

  • 1949年

    • 4月，人民解放军占领南京

    • 9月，中国人民政治协商会议第一届全体会议召开，标志着新民主主义革命的基本胜利
